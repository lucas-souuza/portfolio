# Portfólio Pessoal

Portfólio pessoal feito com HTML + Tailwind CSS (via Tailwind CLI).  
O objetivo do site é me apresentar e mostrar como posso agregar às empresas por meio dos conhecimentos desenvolvidos na faculdade e em projetos pessoais.
Ademais, foi desenvolvido com padrão profissional e atenção a detalhes, com apoio de ferramentas de IA para acelerar iteração e refinamento.
Revisão e decisões finais foram feitas por mim

## Demo
- GitHub Pages: https://lucas-souuza.github.io/portfolio/

## Tecnologias
- HTML
- Tailwind CSS (CLI)
- Node.js / npm

## Estrutura do projeto
- `index.html` — página principal (usa classes do Tailwind).
- `src/input.css` — arquivo de entrada do Tailwind (`@import "tailwindcss";`). [web:224]
- `assets` — Pasta para guardar alguns arquivos utilizados no projeto e ao CSS do Tailwind CLI.
- `assets/tailwind.css` — CSS gerado automaticamente pelo Tailwind CLI (não editar manualmente).
- `package.json` — dependências e scripts.
